# Examples

This folder contains complete examples that users can copy and modify.

Examples should show a full research workflow: design question, runnable setup, files to modify, expected output, sample data, screenshots, and analysis link.

## Who Should Use This

Use this folder if you want to start from a complete example rather than from a guided exercise. Researchers adapting the framework for a new study should usually begin here.

## What Should Be Added Later

- Human-only examples for minimal two-player choice, public goods, coordination, and social learning
- LLM-only examples for public goods and social learning simulations
- Mixed human-LLM examples for public goods and social learning
- Example-specific configs, sample data, screenshots, and analysis links

## What Not To Put Here

Do not put one-off tutorial instructions that only make sense during the live workshop. Those belong in `hands_on/`. Do not duplicate shared game logic here; examples should import or reference `shared/game_logic/` when possible.

## Completion Standard

Each example should be runnable and should explain its research question, design, how to run it, what files to modify, what output to expect, and how it connects to the tutorial.
