# Slides

This folder contains teaching materials for the three-hour tutorial: the main slide deck, speaker scripts, conceptual figures, and presenter notes.

## Who Should Use This

Use this folder if you are teaching the workshop, adapting the tutorial for a shorter talk, or creating figures for a paper or methods supplement.

## What Should Be Added Later

- `tutorial_slides.pptx` and `tutorial_slides.pdf`
- A full three-hour speaker script
- A shorter 30-minute version
- Conceptual diagrams for the design framework
- A slide-to-repository map that tells participants which folder to open

## What Not To Put Here

Do not put runnable experiment code, raw exported data, API keys, or analysis scripts here. Code belongs in `engines/`, reusable logic belongs in `shared/`, and analysis belongs in `analysis/`.

## Completion Standard

The slides should teach intuition, not code details. Every code-heavy section should point to `hands_on/` or `examples/`.
