# Mixed Public Goods

This folder is reserved for the Empirica-facing mixed public goods example.

The planned flow is:

1. Human participants submit contributions through Empirica.
2. Mock or real LLM agents submit compatible contribution actions through the
   mixed bridge.
3. The shared public goods logic computes payoff and group feedback.
4. Human and LLM actions are exported together with `actor_type`.

The first runnable version should use mock LLM agents by default.
