# Social Learning

This folder is reserved for the human-only Empirica social learning example.

The example should import logic from `shared/game_logic/social_learning/` and
use Empirica for participant timing, treatment assignment, and browser display.

Expected first implementation:

- Repeated estimates from human participants.
- A control condition with only own previous estimate.
- A social-information condition with the previous group average.
- Export rows compatible with the shared action and observation schemas.
