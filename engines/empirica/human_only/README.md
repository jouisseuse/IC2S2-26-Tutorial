# Human-Only Empirica Examples

This folder is for experiments where all actors are human participants using
Empirica in the browser.

Current first-version example:

- `minimal_two_player_choice/`: two players, one round, one decision, one
  feedback screen.

Planned examples:

- `public_goods/`: group contribution game using shared public goods logic.
- `social_learning/`: repeated estimates with optional social information.

Do not put mock LLM agents or model adapter code here. Mixed experiments belong
in `engines/empirica/mixed_human_llm/` or `engines/mixed_human_llm/`.
