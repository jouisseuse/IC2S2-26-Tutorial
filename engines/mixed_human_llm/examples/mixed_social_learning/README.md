# Mixed Social Learning

This folder will contain the complete mixed human-LLM social learning example.

It should combine:

- Empirica human estimates.
- Mock LLM estimates using the same observation schema.
- The shared social learning game logic.
- Exports that support comparison across human-only, LLM-only, and mixed modes.

Until the full example is built, use the bridge files one level up as the
reference implementation pieces.
