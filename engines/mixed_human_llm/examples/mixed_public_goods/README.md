# Mixed Public Goods

This folder will contain the complete mixed human-LLM public goods example.

It should combine:

- Empirica human action collection.
- Mock LLM agents from `engines/llm_simulation/agents/`.
- The shared public goods game logic.
- One shared action/event export with `actor_type` set to `human` or `llm`.

Until the full example is built, use the bridge files one level up as the
reference implementation pieces.
