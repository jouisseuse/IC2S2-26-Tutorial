# Assets

This folder contains visual and non-code materials used by the tutorial and documentation.

Examples include diagrams, screenshots, short demo videos, and worksheet templates.

## Who Should Use This

Use this folder if you are preparing workshop materials, documenting expected output, creating fallback materials for participants with setup problems, or adding visual material for a paper supplement.

## What Should Be Added Later

- Diagrams for the design framework
- Screenshots of the admin panel and participant windows
- Short demo videos for minimal experiment, public goods, LLM agents, and mixed human-LLM experiments
- Design worksheet in PDF and editable formats

## What Not To Put Here

Do not put source code, raw logs, API keys, or analysis notebooks here. Code belongs in `engines/`, logs and sample data belong in `analysis/`, and notebooks belong in `analysis/notebooks/`.

## Completion Standard

Tutorial participants should be able to follow along using screenshots, videos, and worksheets even when local setup fails.
