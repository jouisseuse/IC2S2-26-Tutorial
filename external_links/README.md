# External Links

This folder contains curated resources for continued learning.

Each link entry should include a title, short description, why it is useful, and suggested audience.

## Who Should Use This

Use this folder if you are a participant looking for follow-up reading, a collaborator collecting references, or an instructor preparing background material.

## What Should Be Added Later

- Official Empirica documentation
- Official Empirica examples
- Multiplayer experiment readings
- Public goods readings
- Social learning readings
- Human-AI experiment readings
- LLM agent tools

## What Not To Put Here

Do not put copied PDFs, copyrighted articles, raw notes without context, or implementation code here. Keep entries as curated links with short explanations.

## Completion Standard

Participants should be able to continue learning after the tutorial and understand why each linked resource is useful.
